Arghhh. Turns out its my first encounter of the Galileo variant differing from the standard arduino behaviour. Luckily it didn't take long to google up an answer and I got to the bottom of the issue by trying out this with further details also here. If clicking links isn;t your thing, the following is an example of how to run random in a loop:

int randNumber;
int small = 0;
int big = 10;

void setup()
{
  Serial.begin(9600);
}

void loop()
{
  randNumber = small + (random() % (big - small));
  Serial.println(randNumber);
}
